
Принцип работы системы:
Кадры с камеры обрабатываются при помощи входных фильтров (цвета и контрастности) для выделения интересующих областей изображения для последующего поиска в них заданных объектов. Далее эти области масштабируются до размеров, заданных в параметрах входного слоя нейросети. Затем нейросеть определяет класс объекта (дорожный знак, автомобиль и т.п.) и выдает индикацию его наличия на исследуемой области. Затем на исходное изображение покадрово накладываются значки, идентифицирующие найденные объекты.
